# Datorika Portfolio

A static HTML portfolio website showcasing computing work done in 10th grade (written in Latvian).

## Project Structure

- `index.html` - Main landing page
- `otrais.html` - Secondary page
- `style.css` - Custom stylesheet
- `image/` - Media assets (GIF, PNG, MP4, STL files)

## Technologies

- Plain HTML5, CSS3, vanilla JavaScript
- W3.CSS framework (via CDN)
- Google Fonts (Poppins)

## Running the Project

The project is served using Python's built-in HTTP server on port 5000:

```
python3 -m http.server 5000 --bind 0.0.0.0
```

## Deployment

Configured as a static site deployment with the root directory as the public directory.
